🧠 DBMS (Database Management System) – Basics

📌 1. Database
A database is an organized collection of data.
It stores data in a structured format (tables).

Example:
Roll No | Name  | Age
101     | Riya  | 20
102     | Aarav | 21

---

📌 2. DBMS Definition
DBMS is software used to:
- Store data
- Manage data
- Retrieve data
- Update data efficiently

Examples:
MySQL, PostgreSQL, Oracle, MongoDB

---

📌 3. Why DBMS is needed (Problems in File System)

❌ File System Problems:
- Data redundancy (duplicate data)
- Data inconsistency
- Slow searching
- No security control
- No backup/recovery
- No relationships between data

---

📌 4. Advantages of DBMS

✅ Reduces redundancy
✅ Maintains data consistency
✅ Provides security (access control)
✅ Fast data retrieval
✅ Backup and recovery
✅ Supports relationships between data

---

📌 5. Real-Life Analogy

File System = messy cupboard (no order)
DBMS = organized wardrobe (properly arranged and labeled)

---

📌 6. Key Purpose of DBMS
To store, manage, and retrieve data efficiently with security and structure.

---

📌 7. Where DBMS is used
- Banking systems
- Social media apps (Instagram, YouTube)
- E-commerce (Amazon, Flipkart)
- College management systems
